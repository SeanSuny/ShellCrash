功能简介：
--

~**Docker run部署：**<br>

```shell
docker run -dit \
--name ShellCrash \
--hostname ShellCrash \
--network bridge \
--restart always \
-p 9999:9999 \
-v /opt/shellcrash:/etc/ShellCrash \
seansuny/shellcrash:latest
```

设备支持：
--

~支持CPU架构AMD64、ARM64两种架构<br>
